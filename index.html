Rules

Le jeu est simple, vous devez choisir entre Pierre, Feuille ou Ciseaux. L'ordinateur fera de même. Le gagnant est déterminé par les règles suivantes :

  • Pierre bat Ciseaux
  • Ciseaux bat Feuille
  • Feuille bat Pierre

Si les deux parties ont fait le même choix, personne de gagne de point.

Le jeu se joue en 5 manches, le joueur avec le plus de points à la fin des 5 manches gagne la partie.
